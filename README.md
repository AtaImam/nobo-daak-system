# 🚀 Nobo-Daak (Secure Digital Mailbox)

## 📌 Overview

**Nobo-Daak** is a secure digital communication and document management platform designed to modernize Bangladesh’s traditional paper-based system.

It provides a centralized system where users can receive, manage, and interact with official documents such as bills, notices, payslips, and exam information in a secure and efficient way.

The system addresses real-world problems like delayed postal services, lost documents, and fragmented communication by offering a unified digital solution.

📌 *Presented at a university project showcase*

---

## 🎯 Objectives

* Create a centralized digital mailbox for users
* Replace manual communication with automated digital processes
* Enable online bill payment with instant receipt generation
* Provide real-time notifications and alerts
* Ensure transparency through activity logs
* Maintain high-level data security and privacy
* Deliver analytical insights through dashboards

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Integration:** REST APIs (Bkash, Nagad, Bank, Card payments)
* **Other:** Courier tracking API, Notification system

## ⚙️ Key Features

### 🔐 User Management

* Secure registration and login system
* Profile management with personal data updates

* Centralized inbox for official and personal documents

### 📊 Dashboard & Analytics

## 🗄️ Database Design

The system uses a relational database with key entities such as:

* Users
* Companies
* Documents
* Invoices
* Payments
* Receipts
* Activity Logs

Relationships ensure:

* One user can have multiple documents and invoices
* Each payment generates a receipt
* All user actions are recorded for transparency

## 🔄 System Workflow

The system integrates multiple entities:

* Users

* Payment Gateways
* Notification Services

➡️ Users receive documents, pay bills, and get alerts
➡️ External systems send invoices, notices, and updates
